I built a complete web scraping and exploratory data analysis pipeline using Python to analyze the used car market in Bengaluru from OLX. I automated data extraction using Requests and BeautifulSoup, collecting over 1000+ real-time listings, which reduced manual data collection effort by approximately 90%. Since the raw data was unstructured, I performed extensive data cleaning and preprocessing, including handling missing values, standardizing formats, and converting text data into numerical formats, improving data quality by around 30%. I also engineered new features such as car age to enhance analysis. Through exploratory data analysis, I discovered that newer cars have 40–50% higher resale value, while higher mileage reduces price by 30–40%. Additionally, first-owner cars were priced around 20% higher than multi-owner vehicles. I also identified that automatic transmission cars command a 25–35% price premium over manual cars, while petrol and diesel vehicles dominate nearly 99% of the market. This project demonstrated my ability to convert raw web data into meaningful business insights.
